# Student To-Do List DevOps Project

This project is a simple Student To-Do List web application developed using Python Flask.

The project demonstrates a DevOps-based quality assurance pipeline using:

- GitHub
- GitHub Actions
- Selenium
- SonarQube Cloud
- Docker

## Application Features

- Add a student task
- View all tasks
- Mark a task as completed
- Delete a task

## Project Purpose

The purpose of this project is to demonstrate how DevOps tools can support software development through automation, testing, code quality analysis, and containerization.
